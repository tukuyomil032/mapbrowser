# コマンド一覧 (ja-jp)

ベースコマンド:

- /mapbrowser
- /mb

## 画面管理

- /mb create <w> <h> [name]
- /mb destroy
- /mb list
- /mb info

## ブラウザ操作

- /mb open <url>
- /mb back
- /mb forward
- /mb reload
- /mb fps <value>
- /mb exit

## アイテム付与

- /mb give <pointer|back|forward|reload|url-bar|scroll-up|scroll-down>

## 管理系

- /mb admin status
- /mb admin stop <screenId>

## 注意

- 多くのコマンドはプレイヤー実行前提です。
- URL はセキュリティ設定で検証されます。
